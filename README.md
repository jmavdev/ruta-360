# Ruta 360 - Plataforma de Logística Inversa 🚛

Sistema integral de "Logística en transporte" diseñado para optimizar los retornos de carga de los transportistas, eliminando los viajes en vacío y maximizando la rentabilidad de cada kilómetro. Este repositorio contiene el prototipo Frontend (UI/UX) de la aplicación.

## ✨ Características Principales (Perfiles)

- **Chofer (Mobile-First):** Dashboard de ruta, Check List de salida, centro de incidencias y mapas.
- **Cliente (Desktop/Tablet):** Seguimiento operativo, listado de pedidos y facturación.
- **Maestro Empresa (Desktop):** Dashboard financiero, planificador de rutas, gestión de flota (vehículos) y personal (choferes).
